## Opdracht
<br>

We maken een 2de versie van vorige oefening 'Woorden wisselen'.
Vraag terug een tekst aan de gebruiker die bestaat uit **exact twee woorden** gescheiden door één spatie.  
Wissel de volgorde van de woorden om met behulp van de split()-methode en beeldt de nieuwe tekst af.

### Wat moet je doen?

+ Lees de tekst in.
+ Splits de tekst met behulp van de split()-methode.
+ Wissel de volgorde van de woorden.
+ Druk de nieuwe tekst af.

### Voorbeeld invoer

    Hallo wereld

### Verwachte uitvoer  
    
    wereld hallo
