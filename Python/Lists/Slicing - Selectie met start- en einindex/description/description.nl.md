## Opdracht
<br>

Je krijgt een **fruitmand** als lijst van strings.  
De gebruiker geeft een **startindex** en een **eindindex**.  
Gebruik slicing om een selectie uit de lijst te maken.

### Gegeven

De fruitmand wordt **vast** in het programma gezet als:

fruitmand = ["appel", "banaan", "kiwi", "peer", "mango", "druif", "kers"]  

### Wat moet je doen?  

+ Lees een startindex (int) in.
+ Lees een eindindex (int) in.
+ Maak een nieuwe lijst resultaat met slicing: fruitmand[start:einde]
+ Druk resultaat af.

### Verwachte uitvoer

De uitvoer bestaat uit **exact één regel**:
+ de lijst resultaat

### Voorbeeld van uitvoer  
    
    ['banaan', 'kiwi', 'peer']
