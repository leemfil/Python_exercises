## Opdracht
<br>

Je krijgt een **fruitmand** als lijst van strings.  
Gebruik **slicing** om verschillende delen uit deze lijst te selecteren.

### Gegeven

De fruitmand wordt **vast** in het programma gezet als:

fruitmand = ["appel", "banaan", "kiwi", "peer", "mango", "druif"]  

### Wat moet je doen?  

+ Maak een nieuwe lijst middelste_fruit met het **3de, 4de en 5de** stuk fruit.
+ Maak een nieuwe lijst zonder_randen zonder het **eerste en laatste** fruit.
+ Maak een nieuwe lijst laatste_deel die **start bij `"peer"` en loopt tot het einde**.
+ Druk de drie lijsten af, **elk op een aparte regel**, in bovenstaande volgorde.

### Verwachte uitvoer

De uitvoer bestaat uit **exact drie regels**:
+ op de eerste regel: de lijst middelste_fruit
+ op de tweede regel: de lijst zonder_randen
+ op de derde regel: de lijst laatste_deel

### Voorbeeld van uitvoer  
    
    ['kiwi', 'peer', 'mango']  
    ['banaan', 'kiwi', 'peer', 'mango']  
    ['peer', 'mango', 'druif']
