### Opdracht

Schrijf een programma dat:

- de som van de 3 gegeven gehele getallen berekent endeze waarde toekent aan een variabele (bv. som).
- de berekende waarde op het scherm afdrukt.

### Voorbeeld

**Gegeven**

Drie gehele getallen g1, g2 en g3.

**Uitvoer**

     De som is 25
